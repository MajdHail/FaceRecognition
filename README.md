# Face Recognition

Real-time face direction detection using Python, OpenCV, and MediaPipe.

## Features
- Detects face with orientations on the X-axis.
- Works on macOS with Python 3.11.
- Uses MediaPipe FaceMesh for accurate and stable detection.

## Installation
```
git clone https://github.com/MajdHail/FaceRecognition.git
cd FaceRecognition
pip install -r requirements.txt
```
